# Currency Converter

A simple web-based currency converter application built with HTML, CSS, and JavaScript.

## Features

- Convert between different currencies using real-time exchange rates
- Modern glassmorphism UI with dark gray theme
- Responsive design for mobile and desktop
- Smooth animations and transitions

## Usage

1. Open `index.html` in your web browser
2. Select the currencies you want to convert from and to
3. Enter the amount and see the converted value

## Technologies Used

- HTML5
- CSS3 (with modern features like backdrop-filter)
- JavaScript (ES6+)

## Setup

No setup required - just open the HTML file in a browser. Ensure you have an internet connection for fetching exchange rates.
